publication: '*Reuse of Off-the-Shelf Components*'
abstract: Meshing tools are traditionally built in a one by one basis without reusing
  already developed parts. However, there are several concepts within this application
  domain that are present in most tools. Developing software components implementing
  these concepts is extremely time consuming and requires highly specialized programmers.
  Software product lines is a way of systematically reusing assets. We propose a layered
  product line architecture for meshing tools. We specify it formally using xADL,
  and we show that it fits some already built tools.
